# yolov8peoplecounter
use yolov8 for people counter

## Installation
### 1. Installing and run
Step1. Install yolov8peoplecounter.
```shell
conda create -n py38 python=3.8
conda activate py38
pip install opencv-python pandas ultralytics cvzone
```

Step2. run 

```shell
python main.py
```
